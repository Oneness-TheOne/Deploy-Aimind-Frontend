아이마음 센터 이용 가이드 (RAG 텍스트)

1) 서비스 개요
아이의 그림을 AI로 심리 분석하고, 맞춤 솔루션과 상담소 정보를 제공하는 서비스입니다. 주요 메뉴는 그림 분석, 그림일기 OCR, 맞춤 솔루션, 주변 상담소, 맘스퀘어(커뮤니티), 마이페이지입니다.

2) 주요 기능 요약
- 그림 분석: 그림 업로드 또는 직접 그리기 후 아이 정보를 입력하면 AI가 분석하고 결과 리포트를 제공합니다.
- 그림일기 OCR: 그림일기 이미지의 손글씨를 텍스트로 변환합니다.
- 맞춤 솔루션: 치료법 제안, 색상 추천, 가정 활동 가이드 제공.
- 주변 상담소: 지도/리스트 뷰, 검색, 필터, 정렬 제공.
- 맘스퀘어(커뮤니티): 게시글 탐색, 카테고리/태그/검색, 좋아요/북마크, 글쓰기.
- 마이페이지: 분석 기록, OCR, 아이 관리, 활동 내역, 저장된 게시글.

4) 그림 분석 사용 흐름
1. 그림 입력: 업로드 또는 직접 그리기 탭 선택.
2. 아이 정보 입력: 이름(별명), 그림 유형, 나이, 성별.
3. 분석 시작: 업로드 이미지 + 나이 + 그림 유형이 있어야 시작 버튼 활성화.
4. 분석 진행: 구성요소 분석 → 발달 단계 평가 → 또래 비교 → 종합 분석.
5. 결과 확인: 종합 점수, 발달 단계, 정서 상태, 요약 및 탭별 리포트 확인.

5) 그림일기 OCR 사용 흐름
- 그림일기 이미지를 업로드하면 텍스트로 변환됩니다.
- 정확도 팁: 밝은 곳에서 촬영, 그림자 피하기, 수평 맞추기, 흔들림 없이 선명하게 촬영.

6) 맞춤 솔루션 구성
- 치료법 제안: 미술/놀이/독서 치료 활동과 단계 제공.
- 색상 추천: 감정 상태별 추천 색상과 활동.
- 활동 가이드: 가정에서 할 수 있는 활동과 준비물/기대 효과 안내.

7) 주변 상담소 찾기
- 지도/리스트 뷰 전환 가능.
- 검색(센터명/주소), 필터(전문 분야), 정렬(거리/평점).
- 센터 정보: 주소, 전화, 영업시간, 평점, 리뷰 수, 거리, 전문 분야.

8) 커뮤니티(맘스퀘어)
- 카테고리: 자유게시판/육아 꿀팁/Q&A/상담 후기/전문가 칼럼.
- 검색 및 인기 태그로 탐색 가능.
- 게시글 상세에서 조회수, 댓글수, 좋아요, 북마크 확인.

9) 로그인/회원가입
- 로그인: 이메일/비밀번호, 로그인 상태 유지 옵션.
- 회원가입: 이름/이메일/비밀번호/자녀 연령대/약관 동의.

10) 마이페이지
- 개요: 발달 영역별 분석 요약, 최근 분석 기록.
- 분석 기록: 히스토리 목록과 PDF/공유 버튼.
- 그림일기 OCR 탭 제공.
- 아이 관리: 등록 아이 목록.
- 활동/저장됨: 커뮤니티 활동 내역과 북마크 게시글.

11) 챗봇 안내 문구
- 시작 메시지: “안녕하세요! 아이마음 상담 도우미입니다. 아이의 그림 심리 분석이나 서비스 이용에 대해 궁금한 점이 있으시면 편하게 물어보세요.”
- 고지: “AI 상담 도우미는 참고용이며, 전문 상담을 대체하지 않습니다.”
